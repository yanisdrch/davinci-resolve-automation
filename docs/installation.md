# Installation

It's very simple to install all the scripts! You just have to clone the Git repository into DaVinci Resolve's `Utility` folder. In DaVinci, every scripts will appear in  `Workspace/Scripts/davinci-resolve-automation/scripts`

## Utility Folder Paths

- Windows : `C:\ProgramData\Blackmagic Design\DaVinci Resolve\Fusion\Scripts\Utility\`
- mac : `~/Library/Application Support/Blackmagic Design/DaVinci Resolve/Fusion/Scripts/Utility/`
- Linux : `~/.local/share/DaVinciResolve/Fusion/Scripts/Utility/`

## Cloning the Repository

Open a terminal in the Utility folder and  run

```bash
git clone https://github.com/yanisdrch/davinci-resolve-automation.git
```
